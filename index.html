The Mandelbrot Set Formula

zn+1 = zn2 + c

Explanation: Start with z0 = 0. For each point c in the complex plane, repeatedly apply the formula. If the value stays bounded (doesn't go to infinity), the point is in the Mandelbrot set (black). Otherwise, the number of steps before it escapes determines the color.
